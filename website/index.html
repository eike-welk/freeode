Intro

The Freeode project provides a simple programming language (Siml), and a matching compiler, to solve differential equations numerically. The compiler produces Python source code, that uses the SciPy libraries for numerical computations. Currently only ODE can be solved.

The generated Python code can be used with a minimal main program, or in an interactive Python session. With a Python IDE like Eric one can have both at once, very much like the Matlab IDE (which is for the commercial language Matlab).

Installation

Windows

The development happens on Linux and no attempt has been made to port the compiler to Windows. In principle however, everything should work on Windows. The Python language and necessary libraries ( Boost, SiPy) are all available for Windows. So you're on your own there.

If you have a C++ compiler for Windows and some time, you could probably port the compiler to Windows with minimum effort. The Python part needs no change, since Python promises to be platform independent. I would be very glad to receive patches and binaries.
